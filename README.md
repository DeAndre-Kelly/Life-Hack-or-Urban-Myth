The Purpose of the Life Hack or Urban Myth quiz app:
* The idea behid this app is to use quiz based questions to help users study and better identify real hacks
  versus myths, the quiz tallies up the score of your answers as well as provides a review page to help the
  user review their efforts and as well to help them understand where they've went wrong and help guide them
  by not only providing the information but by actually helping them put their heads to it.

* The design is simple and easy to use, it provides a straight forward and simple user interface and clear
  indication of feedback as well as instructions.

The utilisation of Github and Github actions:
* Github assisted greatly in reducing errors due to the automated testing and was much easier to maintain.
